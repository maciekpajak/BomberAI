# BomberAI
 
Bomberman clone written in python using pygame. 
 
# AI in games - University project

## Setup
1. Create virtual env with your favourite tool
2. Activate it
3. To install required dependencies (based on your system) run
```
python install.py
```

## Run 

Run using menu file:
```
python menu.py 
```

Run Q-learning:
```
python qlearning.py 
```
